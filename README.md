# Heroes

My personalnode and react playground

## Installation

```
git clone <repo>

cd <cloned repo dir>

npm install
```
## Starting a server

Quickest method is to use (uses babel-node to build on the fly):

```
npm run start
```

Alternative method is to:

```
npm run build

npm run serve
```

Then browse to [http://localhost:3000](http://localhost:3000)
